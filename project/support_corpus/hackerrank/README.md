## HackerRank Corpus Folder

Approved source:

- https://support.hackerrank.com/

Put extracted HackerRank support content in this folder only.

Recommended file topics:

- account access
- assessments
- submission flow
- supported languages
- navigation and workflows

Good example filenames:

- `account-access.md`
- `assessment-workflows.md`
- `submission-help.md`

Do not place Claude or Visa content here.
