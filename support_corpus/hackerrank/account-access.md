## Extracted support content

- HackerRank documents that users can change their password from the account settings area after signing in.
- The documented flow is: sign in, open the profile menu, open `Settings`, enter the current password plus the new password, then save.
- For forgotten passwords, the documented path is to use `Forgot Password?` on the login page, enter the work email address tied to the account, and follow the reset link sent by email.
- This source supports safe replies about basic password reset and password update workflow.
- This source does not support promising account restoration, access grants, or admin-side permission changes.
